HTML Structure

1.หา Attribute ของ tag ต่อไปนี้

No. Tag Name Attribute ใช้งานอย่างไร
1. h id,class,style คำสั่งในการตั้งชื่อหัวข้อ
2. p id,class,style คำสั่ั่งที่สร้างย่อหน้า
3. div id,class,style คำสั่งสร้างกล่องข้อความ
4. img id ,class ,style ,width ,height ,src ,alt คำสั่งในเรียกรูปภาพ
5. a id ,class ,style ,href ,target คำสั่งสำหรับลิ้งไปยังข้อมูลหน้าอื่น
6. ul id,class,style คำสั่งในการสร้างหัวข้อย่อยแบบไม่มีตัวอักษร
7. ol id,class,style,value คำสัื่งในการสร้างหัวข้อย่อยแบบมีตัวเลข
8. table id ,class ,style คำสั่งสร้างตาราง
9. form id ,class ,style ,action ,method ,name คำสั่งสร้างแบบฟอร์ม
10. button id ,class ,style ,disabled ,name ,type ,value คำสั่งสร้างปุ่ม

2.ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

No. เปรียบเทียบ ความเหมือน ความต่าง
1. <hr> & <br> ขึ้นบรรทัดใหม่ hr คือ เส้นแบ่งหน้ากระดาษ
br คือ ขึ้นบรรทัดใหม่อย่างเดียว
2. <ul> & <ol> คำสั่งในการสร้างหัวข้อย่อย หัวข้อย่อยที่สร้างเป็นแบบมีตัวอักษรและไม่มีตัวอักษร
3. <select> & <checkbox>
& <radio button>		 ตัวเลือกในแบบฟอร์ม select เลือกได้หลายรายการ และแสดงผลแบบเห็นแค่ตัวเลือกเดียวได้
checkbox สามารถเลือกได้หลายรายการ
radio button เลือกได้แค่รายการเดียว
4. <div> & <span> สร้างกล่องข้อความ div ใช้ในการรวมกลุ่มข้อมูลของแถวในตารางและตำแหน่งของเพจ
span ใช้ในการเปลี่ยนสไตล์บน webpage โดยไม่ส่งผลต่อข้อความข้างใน
5. <script> & <style> ใช้ในการกำหนดชุดคำสั่ง script แท็กถูกใช้เพื่อกำหนดชุดคำสั่ง JavaScript
style แท็กใช้ในการกำหนดชุดคำสั่ง CSS

3.ให้หาว่า input type แต่ละ type แตกต่างกันอย่างไร?

No. Input Type ความแตกต่าง และลักษณะการใช้งาน
1. button เป็นรูปแบบปุ่มกด
2. checkbox เลือกติ้กเครื่องหมายถูกต้อง
3. color เป็นรูปแบบการเลือกสี
4. date เลือกใส่ข้อมูลวันเดือนปี
5. email สำหรับใส่อีเมล์
6. file สำหรับแนบไฟล์ข้อมูล
7. hidden การซ่อนเพื่อไม่ให้มองเห็น
8. image เป็นการใส่รูปภาพ
9. month การเลือกหรือใส่ข้อมูล เดือน ปี
10. number การใส่ข้อมูลด้วยตัวเลข
11. password การใส่รหัสจะมองไม่เห็นข้อความที่กำลังใส่
12. radio เลือกที่ต้องการจะเลือกเพียงช่องเดียว
13. range การเลือกช่วงที่ต้องการโดยเลือกจากจุดเริ่มต้นและจุดสิ้นสุด
14. reset การรีเซทข้อมูลใหม่อีกครั้ง
15. search การค้นหาสิ่งที่ต้องการ
16. submit ตกลงหรือยืนยัน
17. tel ช่องกรอกเบอร์โทรศัพท์
18. text กรอกข้อความตัวอักษร
19. time กรอกข้อมูลในรูปแบบเวลา
20. url ที่อยู่ของลิ้งที่ต้องการนำไป
21. week เป็นการเลือกข้อมูลในรูปแบบสัปดาห์ วัน ปี
