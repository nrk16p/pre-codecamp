HTML structure


Tag Name Attibute (support HTML5) ใช้งานอย่างไร
h id ,class ,style แท็กที่ใช้ในการกำหนดส่วนหัว HTML
p id ,class ,style แท็กกำหนดย่อหน้า Paragraph
div id ,class ,style แท็กที่สามารถครอบแท็กส่วนต่างๆได้ หรือส่วนในเอกสารต่างๆ
img id ,class ,style ,width ,height ,src ,alt แท็กกำหนดภาพ
a id ,class ,style ,href ,target แท็กกำหนดไฮเปอร์ลิงก์ซึ่งใช้เพื่อลิงก์จากหน้าหนึ่งไปอีกหน้าหนึ่ง
ul id ,class ,style แท็กกำหนดรายการ (สัญลักษณ์แสดงหัวข้อย่อย)
ol id ,class ,style ,value แท็กกำหนดรายการ (สัญลักษณ์ตัวเลขหรือตัวอักษร)
table id ,class ,style แท็กกำหนดตาราง
form id ,class ,style ,action ,method ,name แท็กจะใช้ในการสร้างรูปแบบการป้อนข้อมูล
button id ,class ,style ,disabled ,name ,type ,value แท็กกำหนดปุ่มที่คลิกได้

เปรียบเทียบ Tag ความเหมือน ความแตกต่าง
<hr> & <br> ขึ้นบรรทัดใหม่ hr คือ เส้นแบ่งหน้ากระดาษ
br คือ ขึ้นบรรทัดใหม่อย่างเดียว
<ul> & <ol> แท็กกำหนดลิสรายการ ul มีสัญญาลักษณ์เป็นจุด
ol มีสัญญาลักษณ์เป็นตัวเลขตามลำดับ
<select> & <checkbox>
& <radio button>		 ตัวเลือก ในหน้าฟอร์ม select เลือกได้หลายรายการ และแสดงผลแบบเห็นแค่ตัวเลือกเดียวได้
checkbox สามารถเลือกได้หลายรายการ
radio button เลือกได้แค่รายการเดียว
<div> & <span> สามารถครอบแท็กอื่นๆได้ div ครอบแล้วขึ้นบรรทัดใหม่
span ครอบแล้วสามารถอยู่ในบรรทัดเดียวกันได้
<script> & <style> แท็กที่ใช้ในการกำหนดชุดคำสั่ง script แท็กถูกใช้เพื่อกำหนดชุดคำสั่ง JavaScript
style แท็กใช้ในการกำหนดชุดคำสั่ง CSS

input type ความแตกต่าง และลักษณะการใช้งาน
button เป็นรูปแบบปุ่มกด
checkbox เป็นช่องติกเลือกได้หลายรายการ
color เป็นรูปแบบการเลือกสี
date เป็นรูปแบบการเลือกหรือใส่ข้อมูล วัน เดือน ปี
email เป็นช่องกรอกข้อมูลอีเมล
file เป็นช่องเลือกไฟล์ หรือการแนปไฟล์
hidden เป็นรูปแบบการซ้อนให้มองไม่เห็น
image เป็นรูปแบบการแสดงภาพ
month เป็นรูปแบบการเลือกหรือใส่ข้อมูล เดือน ปี
number เป็นรูปแบบการเลือกหรือใส่ข้อมูลตัวเลข
password เป็นรูปแบบการเลือกหรือใส่ข้อมูล password ข้อมูลที่กรอกจะแสดงเป็นเครื่องหมาย *
radio เป็นช่องติกเลือกรายการได้เพียงรายการเดียว
range เป็นรูปแบบการเลือกระยะที่กำหนด เริ่มต้นที่-สิ้นสุดที่
reset เป็นปุ่มรีเซตข้อมูลในหน้าฟอร์ม
search เป็นช่องกรอกข้อมูลค้นหา
submit เป็นปุ่มกดเพื่อตกลงหรือยืนยันการทำงาน
tel ช่องกรอกข้อมูลเบอร์โทรศัพท์
text ช่องกรอกข้อมูลแบบเป็นตัวอักษร
time เป็นช่องกรอกข้อมูลหรือแบบเลือกข้อมูล เวลา
url ช่องกรอกข้อมูลแบบกำหนดชื่อเว็บแอดเดรส
week เป็นช่องกรอกข้อมูลหรือแบบเลือกข้อมูล สัปดาห์ วัน ปี



Tag Output คำอธิบาย
<strong> </strong> This is tag strong ทำให้เป็นตัวหนา
<label> </label> This is tag ใช้กำหนดอยู่หน้าฟอร์ม หรือใช้เชื่อมโยงข้อมูลได้ภายได้คำสั่งพิเศษ
<del> </del> This is tag del ทำให้ตัวอักษรโดนขีดกลาง
<b> </b> This is tag b ทำให้เป็นตัวหนา
<em> </em> This is tag em ทำตัวอักษรเอียง
<mark> </mark> This is tag mark ใช้ไฮไลท์ข้อความ โดยมีสีที่พื้นหลังตัวอักษร
<i> </i> This is tag i ทำตัวอักษรเอียง
<sub> </sub> This is tag sub กำหนดให้เป็นตัวอักษรตกบรรทัดลง
<sup> </sup> This is tag sup กำหนดให้เป็นตัวอักษรยกบรรทัดขึ้น
